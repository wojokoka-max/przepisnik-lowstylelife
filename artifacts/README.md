# Artifacts

Ten folder pochodzi z eksportu Replit i zawiera kilka artefaktów projektu.

- `przepisnik-mobile` - aktywna aplikacja mobilna Expo / React Native.
- `api-server` - backend/API.
- `przepisnik` - archiwalny/webowy artefakt Vite/React. Zachowany jako materiał źródłowy i backup logiki.
- `mockup-sandbox` - archiwalne środowisko mockupów.

Najważniejsza zasada: dla publikacji mobilnej pracować przede wszystkim w `przepisnik-mobile`. Katalogów `przepisnik` i `mockup-sandbox` nie usuwać bez osobnej decyzji i backupu.
