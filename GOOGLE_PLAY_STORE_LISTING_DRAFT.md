# Google Play store listing draft

## App title

Preferred product name:

```text
Przepiśnik. Inteligentny organizer kulinarny
```

If Google Play title length rejects it, use:

```text
Przepiśnik
```

and move the rest to the short description.

## Short description draft

```text
Inteligentny organizer kulinarny do przepisów, planowania posiłków i list zakupów.
```

## Full description draft

```text
Przepiśnik pomaga uporządkować codzienne gotowanie: zapisywać przepisy, planować posiłki, tworzyć listy zakupów i korzystać z funkcji AI wtedy, gdy naprawdę przyspieszają pracę w kuchni.

Możesz dodać przepis ręcznie, z linku, ze zdjęcia albo przez dyktowanie. Aplikacja porządkuje składniki, przygotowanie i notatki, żeby przepis był wygodny do odczytania także podczas gotowania.

Najważniejsze funkcje:
- zapisywanie własnych przepisów,
- import przepisu ze zdjęcia,
- dyktowanie treści przepisu,
- generator przepisu na podstawie dania lub składników,
- planner posiłków,
- lista zakupów,
- edycja zapisanych przepisów,
- notatki do przepisów.

Przepiśnik jest tworzony jako spokojne, praktyczne narzędzie do domowej kuchni: bez chaosu, bez przypadkowych rozpraszaczy, z naciskiem na wygodę i estetykę.
```

## Graphics needed

Google Play listing will need real assets from the working mobile app:

- app icon: 512 x 512 px,
- feature graphic: 1024 x 500 px,
- phone screenshots, recommended minimum:
  - login/welcome screen,
  - recipe list,
  - add recipe/manual or photo import,
  - recipe detail,
  - planner,
  - shopping list.

Do not use web screenshots. Use Android mobile screenshots from the real app.

## Review notes

Possible reviewer note:

```text
The app requires an account. Reviewers may create a standard email/password account from the login screen. Admin GitHub login is reserved for the owner and is not required for review.

AI features require internet access and a working production backend.
```

## Monetization note

Do not mention premium, subscriptions or paid profiles in the listing until monetization is implemented and configured in Google Play.
